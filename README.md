
# Parts of Speech 

App get percentage of parts of speech from the uploaded text file


## Installation

clone the repository

For Backend

```bash
  cd api 
  yarn install  
  yarn dev 
```

For Client

```bash
  cd client 
  yarn install  
  yarn start 
```
    
## Run in docker container


```bash
  docker compose-up
```


## API Reference 

#### Upload File and Get Result

```http
  POST /upload
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `file` | `text file` | **Required**.|
